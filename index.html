📁 Projektstruktur

📦 fkui-helix-prototype/ ├── 📁 src/ # Vue-applikation │ ├── 📁 components/ # Vue-komponenter │ ├── 📁 views/ # Vue-vyer │ ├── 📁 css/ # Helix design tokens │ └── 📁 styles/ # SCSS och bridge-filer ├── 📁 scripts/ # Genereringsscript │ ├── generate-helix-fkui-bridge.js │ ├── generate-figma-enhanced-bridge.js │ └── start-vue.js ├── 📁 config/ # Konfigurationsfiler │ ├── vite.config.ts │ ├── tsconfig.json │ └── eslint.config.mjs ├── 📁 docs/ # Dokumentation │ ├── HELIX-ARBETSFLÖDE.md │ └── CONTRIBUTING.md ├── 📁 tools/ # Utvecklingsverktyg └── 📁 demo/ # Demo-filer

🛠️ Kommandon

  • Starta utveckling: npm run dev
  • Generera bridge: node scripts/generate-helix-fkui-bridge.js
  • Bygg för produktion: npm run build
